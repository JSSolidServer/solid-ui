findImage

  • findImage(thing: NamedNode): string
    • Defined in src/widgets/buttons.ts:328

    Find something we have as explicit image data for the thing See also imagesOf

    Parameters

    • thing: NamedNode

      The thing for which we want to find an image

    Returns string

    The URL of a globe icon if thing equals ns.foaf('Agent') or ns.rdf('Resource').
